
IMAP
```bash
# Authenticate user
1 LOGIN user pass

# List mailboxes
1 LIST "" *

# Open inbox
1 SELECT INBOX

# Retrieve full email
1 FETCH <ID> all

# Logout
1 LOGOUT
```

POP3
```bash
# Provide username
USER user

# Authenticate user
PASS pass

# Get mailbox stats
STAT

# List emails
LIST

# Retrieve email
RETR <id>

# Delete email
DELE <id>

# Close connection
QUIT
```

Secure Access

```bash
# Access IMAP over SSL
curl -k 'imaps://<IP>' --user user:pass

# Connect to IMAPS manually
openssl s_client -connect <IP>:imaps

# Connect to POP3S manually
openssl s_client -connect <IP>:pop3s
```
