

```bash
# Scan RDP
nmap -sV -sC <IP> -p3389 --script rdp*

# Detailed packet trace (noisy)
nmap -sV -sC <IP> -p3389 --packet-trace --disable-arp-ping -n

# Connect (Linux)
xfreerdp /u:<user> /p:"<password>" /v:<IP>
```
