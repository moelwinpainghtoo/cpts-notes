
```bash
# Connect and interact with SMTP server
telnet <IP> 25

# Check for open relay vulnerability
nmap -p25 --script smtp-open-relay <IP>
```

### SMTP Commands

```bash
# Start SMTP session
HELO / EHLO

# Authenticate to server
AUTH PLAIN

# Set sender email
MAIL FROM:

# Set recipient email
RCPT TO:

# Send email content
DATA

# Check if user exists
VRFY

# Expand mailing list
EXPN

# Reset session
RSET

# Keep connection alive
NOOP

# Close session
QUIT
```
